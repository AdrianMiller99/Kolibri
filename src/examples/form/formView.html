Simple Kolibri Forms

Kolibri can create MVC forms by using the projector pattern with presentation models.

It only takes a single line of code to create the form, set up the presentation models, and bind all required listeners such that model and view are kept in sync.

The controller contains all required business logic. Only starter.js and formController.js are application-specific.

Models, view, and binding are totally generic. The current form projector only covers "simple" input types (no selects or multiple values).
